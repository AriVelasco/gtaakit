# gtaakit

Arquitectura de referència per a l'automatització de proves d'API.

Treball de Fi de Grau (TFG) - Facultat d'Informàtica de Barcelona (FIB),
Universitat Politècnica de Catalunya (UPC).

Autora: Ariadna Velasco Ibañez
Director: René Serral Gracià
Titulació: Grau en Enginyeria Informàtica (Enginyeria del Software)

## Sobre el projecte

Aquest projecte explora una arquitectura de referència per a
l'automatització de proves d'APIs web modernes. Pren com a punt de partida
el model Generic Test Automation Architecture (gTAA) del programa d'estudi
ISTQB CTAL-TAE v2.0 i el complementa amb l'estil arquitectònic de ports i
adaptadors.

La implementació de referència es desenvolupa en Python amb pytest, httpx i
jsonschema, i es valida sobre una API pública integrada en un pipeline
d'integració contínua.
